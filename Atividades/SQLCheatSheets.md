Shortcuts SQL (atalhos SQL)

1 - Criacao de Tabela:

```
CREATE TABLE <nome-tabela>
  (
    <descricao-atributos>
    <descricao-chaves>
    <descricao-restricoes>
  )
```

1.1 <descricao-atributos> = <nome> + <tipo>

1.2 <tipo> = INTEGER, DECIMAL, NUMERIC, REAL, VARCHAR, CHAR, TEXT, TIMESTAP, DATE, TIME, ETC ...
